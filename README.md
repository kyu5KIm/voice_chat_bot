# 우리엄마 음성 챗봇

![streamlit 실행사진](./img/screen.png)



## 파일
- `app.py`: Streamlit 버전
- `we_mom_voicebot.ipynb`: Jupyter Notebook 버전

## 설치
```bash
pip install -r requirements.txt
```

## 환경변수
`.env` 파일을 만들고 아래처럼 넣어주세요.

```bash
OPENAI_API_KEY=여기에_키_입력
```

## Streamlit 실행
```bash
streamlit run app.py
```
